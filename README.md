# miplist-report
CMVP MIP List Status Report generated from scraping current and archived MIP web pages

## Disclaimer

This is an unofficial tool and is not affiliated with or endorsed by NIST or the Internet Archive. Data is scraped from the [NIST CMVP Modules In Process list](https://csrc.nist.gov/projects/cryptographic-module-validation-program/modules-in-process/modules-in-process-list); historical data is sourced from the [Wayback Machine](https://web.archive.org). This data may not be complete or accurate. Always refer to the official NIST source for authoritative information.
